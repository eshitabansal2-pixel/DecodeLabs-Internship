# Data Classification Using AI:

A machine learning project developed as part of the DecodeLabs AI Internship (Project 2). It uses a Decision Tree Classifier to classify flowers from the Iris dataset.

## Technologies Used:

- Python
- Pandas
- NumPy
- Scikit-learn

## Features:

- Load the Iris dataset
- Split data into training and testing sets
- Train a Decision Tree model
- Predict flower classes
- Evaluate model accuracy

## Files:

- classification.py
- README.md

## How to Run:

1. Install the required libraries.
2. Run:
   python classification.py
