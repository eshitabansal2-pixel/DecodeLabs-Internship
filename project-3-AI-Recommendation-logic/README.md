#  🎯 AI Tech Stack Recommendation System:

##  📌 Project Overview:
This project is an AI-based recommendation system that suggests the top 3 technology career paths based on user's skills.

##  🚀 Features:
- User enters technical skills
- Uses TF-IDF Vectorization
- Uses Cosine Similarity
- Recommends Top 3 career paths
- Displays Match Percentage
- Streamlit Web Application

##  🛠️ Technologies Used:
- Python
- Pandas
- Scikit-learn
- Streamlit
- TF-IDF Vectorizer
- Cosine Similarity

##  📂 Dataset:
The dataset contains technology job roles and their required skills.

##  ▶️ How to Run:

1. Install the required libraries

pip install pandas scikit-learn streamlit


2. Run the application

streamlit run app.py
