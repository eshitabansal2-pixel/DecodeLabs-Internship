# College Assistant Bot:

## Description:
College Assistant Bot is a simple rule-based chatbot built using Python. It helps and guide students by answering common college-related questions.

## Features:
- Greets users
- Provides information about fees
- Gives exam details
- Shows library timings
- Provides attendance information
- Gives internship information
- Provides placement information
- Handles unknown questions with a default response
- Exit command to end the chatbot

## Technologies Used:
- Python 3

## How to Run
1. Install Python 3.
2. Open the project folder.
3. Run the following command:
python bot.py

## Example Questions:
- hello
- hi
- fees
- exam
- library
- attendance
- internship
- placement
- exit
